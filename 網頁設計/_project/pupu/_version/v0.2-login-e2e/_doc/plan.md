# 普普 -- 普男普女社群交友網站

## 想法

1. 像 Threads 一樣的社群軟體，但是主打交友功能。
2. 個人頭像與上傳照片，必須是素顏平日照片，不准強調高帥富，白富美。
3. 個人簡介，必須要真實符合日常生活，不准誇大。
4. 必須填寫徵友條件，如果已婚就寫已婚，不想交友就寫不想交友。
    * 否則就要填寫自己的條件，還有對方的條件
    * 這些條件不准有『高帥富，白富美』，而是： 想不想結婚，生小孩，婚後要和公婆同住嗎？ 要買房嗎？ ...

## 設計

1. 前端用 react，後端用 fastapi，資料庫用 sqlite，通訊用 GraphQL (starlette + graphene)。
2. 資料先用 AI 產生，放在 _data/ 中，包含 users.json 與 posts.json 
    * 要有留言，按讚，轉貼。
3. 介面必須要像 threads 那樣的乾淨簡潔風格。
4. 要有註冊，登入，加好友，發文，回應，私訊 等功能。

## 想法

我覺得現在的社群網站，交友網站，都有 『帥哥美女』 佔優勢的現象
然後每個人都要透過『服裝，化妝，AI.... 』來讓自己外型加分
根本就是外貌協會

所以我想到：
為什麼不用 AI 創建一個『普男普女的社群交友網站』呢？

於是我開始創建：普普 -- 普男普女社群交友網站

在這個網站裡，頭像必須要『素顏』，然後穿『日常生活的服裝』
個人檔案必需要平凡而誠實

徵友條件不准有『高帥富，白富美』，而是： 

> 喜歡什麼，討厭什麼，想要什麼，專長是什麼？想和什麼樣的人交友。
> 住哪個地區，希望和哪些領域的人交流，什麼狀態下會想約見面
> 想不想結婚，生小孩，婚後要和公婆同住嗎？ 要買房嗎？ ...

凡是頭像沒素顏，穿著太突出，強調外表，財富，都會被降觸及 ...

我覺得只有大家真的接受平凡，才能找到朋友和伴侶 ...
