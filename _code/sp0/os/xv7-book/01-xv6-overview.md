# 1. xv6 作業系統概述

## 1.1 什麼是 xv6？

xv6 是一個簡化的 Unix v6 作業系統移植版本，最初由 MIT 教授 Frans Kaashoek 和 Robert Morris 為教學目的開發。它使用 C 語言撰寫，原本執行於 x86 架構，後來被移植到 RISC-V 架構。

xv6 的設計目標是：
- 提供一個簡單但完整的作業系統範例
- 幫助學生理解作業系統的核心概念
- 展示現代作業系統的基礎設施

## 1.2 為什麼要學習 xv6？

1. **代碼量適中**：約 20000 行 C 程式碼
2. **核心概念完整**：涵蓋行程、記憶體、檔案、網路
3. **可執行**：可在 QEMU 中實際運行
4. **教學導向**：每個元件都有清楚的文件說明
5. **真实源码**：不是伪代码，是可以编译运行的真实系统

## 1.3 xv6 的演進

```
xv6 (原始 x86 版本)
    ↓ (2006年 MIT 课程)
xv6 (x86) → xv6-riscv (RISC-V 移植)
    ↓
xv7 (本书版本 - 完整网络与HTTP服务)
```

## 1.4 本書的目標

本書將帶領讀者：
1. 理解 xv6 作業系統的基礎架構與原始碼
2. 學會在 QEMU 中建置與執行 xv7
3. 深入理解作業系統原理（行程、記憶體、檔案、網路）
4. 掌握網路功能擴充的原理與實作
5. 完成 HTTP 伺服器與客戶端的開發
6. 學會自動化測試與除錯技巧

- 已安裝 RISC-V cross-compiler

- 了解基本的 TCP/IP 網路概念
