# 排序演算法

排序是將資料依特定順序排列的演算法。

## 比較排序

### 氣泡排序 O(n²)
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

### 合併排序 O(n log n)
```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)
```

### 快速排序 O(n log n) 平均
```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    return quick_sort([x for x in arr[1:] if x < pivot]) \
         + [pivot] \
         + quick_sort([x for x in arr[1:] if x >= pivot])
```

## 非比較排序

- **計數排序**: O(n + k)
- **桶排序**: O(n + k)
- **基數排序**: O(nk)

## 穩定排序

氣泡排序、合併排序、插入排序是穩定的。
