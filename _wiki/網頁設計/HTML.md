# HTML

HTML (HyperText Markup Language) 是建立網頁的基礎標記語言。

## 基本結構

```html
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <title>網頁標題</title>
</head>
<body>
    <h1>標題</h1>
    <p>段落</p>
    <a href="https://example.com">連結</a>
    <img src="image.jpg" alt="圖片">
</body>
</html>
```

## 常見標籤

| 標籤 | 用途 |
|------|------|
| `<h1>`-`<h6>` | 標題 |
| `<p>` | 段落 |
| `<a>` | 連結 |
| `<img>` | 圖片 |
| `<div>` | 區塊 |
| `<span>` | 行內 |
| `<ul><ol>` | 清單 |
| `<table>` | 表格 |
| `<form>` | 表單 |

## HTML5 新元素

- `<header>`, `<footer>`, `<nav>`
- `<article>`, `<section>`
- `<video>`, `<audio>`
- `<canvas>`, `<svg>`
