# 後端開發

後端處理伺服器端的邏輯、資料庫與 API。

## 常見後端語言

| 語言 | 框架 |
|------|------|
| Node.js | Express, NestJS |
| Python | Django, Flask |
| Java | Spring |
| Go | Gin, Echo |
| Rust | Actix |

## Node.js + Express

```javascript
const express = require('express');
const app = express();

app.get('/api/users', (req, res) => {
    res.json([
        { id: 1, name: 'John' },
        { id: 2, name: 'Jane' }
    ]);
});

app.post('/api/users', (req, res) => {
    const user = req.body;
    // 儲存到資料庫
    res.status(201).json(user);
});

app.listen(3000);
```

## RESTful API 設計

- 資源導向 URI
- 使用 HTTP 方法
- 無狀態
- 回應標準格式
