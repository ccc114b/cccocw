# 響應式設計

響應式設計使網頁能在不同尺寸的裝置上良好顯示。

## viewport meta

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## 媒體查詢

```css
/* 行動優先 */
body {
    font-size: 16px;
}

/* 平板 */
@media (min-width: 768px) {
    body {
        font-size: 18px;
    }
}

/* 桌面 */
@media (min-width: 1024px) {
    body {
        font-size: 20px;
    }
}
```

## 彈性圖片

```css
img {
    max-width: 100%;
    height: auto;
}
```

## 常用斷點

| 裝置 | 寬度 |
|------|------|
| 行動 | < 768px |
| 平板 | 768px - 1024px |
| 桌面 | > 1024px |
