# VSCode

VSCode (Visual Studio Code) 是 Microsoft 開發的程式碼編輯器。

## 基本功能

- **語法高亮**: 多語言支援
- **IntelliSense**: 智慧型程式碼完成
- **偵錯**: 內建偵錯支援
- **Git 整合**: 版本控制
- **擴充**: 豐富的擴充生態

## 常用快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| Cmd+P | 快速開啟檔案 |
| Cmd+Shift+P | 命令面板 |
| Cmd+D | 選取下一個相同文字 |
| Cmd+Shift+L | 選取所有相同文字 |
| Cmd+/ | 註解切換 |
| Alt+Up/Down | 移動行 |

## 擴充推薦

- **Prettier**: 程式碼格式化
- **ESLint**: 程式碼檢查
- **Live Server**: 網頁即時預覽
- **GitLens**: Git 增强
- **Copilot**: AI 輔助程式設計

## 設定

```json
{
    "editor.formatOnSave": true,
    "editor.tabSize": 2,
    "files.autoSave": "onFocusChange"
}
```
