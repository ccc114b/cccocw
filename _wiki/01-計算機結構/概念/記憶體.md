# 記憶體

記憶體是計算機用於儲存資料與指令的硬體元件。

## 記憶體層次

```
┌─────────────────────────────────────────┐
│   暫存器 (Registers)      <- 最快、最小  │
├─────────────────────────────────────────┤
│   快取 (Cache) L1/L2/L3    <- 快、小    │
├─────────────────────────────────────────┤
│   主記憶體 (RAM)            <- 中等     │
├─────────────────────────────────────────┤
│   輔助記憶體 (SSD/HDD)      <- 慢、大   │
└─────────────────────────────────────────┘
```

## 記憶體類型

### 揮發性記憶體
- **DRAM**: 動態隨機存取記憶體，需要定期刷新
- **SRAM**: 靜態隨機存取記憶體，速度更快

### 非揮發性記憶體
- **ROM**: 唯讀記憶體
- **Flash**: 閃存 (USB, SSD)
- **EEPROM**: 可擦除可程式唯讀記憶體

## 記憶體位址

- **位址匯流排**: 傳遞記憶體位址
- **資料匯流排**: 傳遞資料
- **控制匯流排**: 控制讀寫時機
