# 數位邏輯

數位邏輯是計算機硬體的基礎，研究如何使用邏輯閘處理二進位資料。

## 基本邏輯閘

| 閘 | 符號 | 功能 |
|----|------|------|
| AND | ∧ | 輸入都為1輸出為1 |
| OR | ∨ | 輸入任一為1輸出為1 |
| NOT | ¬ | 輸入為0輸出為1 |
| NAND | - | AND 的否定 |
| NOR | - | OR 的否定 |
| XOR | ⊕ | 輸入不同輸出為1 |

## 布林代數

- **交換律**: A + B = B + A
- **結合律**: (A + B) + C = A + (B + C)
- **分配律**: A(B + C) = AB + AC
- **德摩根定理**: ¬(A + B) = ¬A · ¬B

## 組合邏輯

輸出只取決於當前輸入：
- 加法器
- 多工器
- 解多工器
- 編碼器/解碼器

## 時序邏輯

輸出取決於當前輸入與過去狀態：
- 暫存器
- 計數器
- 狀態機
