# Linus Torvalds

芬蘭裔美國軟體工程師，Linux 核心與 Git 版本控制系統的作者。

## 主要貢獻

### Linux 核心 (1991-)

- 開源 Unix-like 作業系統核心
- 成為最廣泛使用的伺服器作業系統
- 催生了 Android 系統

### Git 版本控制 (2005-)

- 分散式版本控制系統
- 成為全球最流行的版本控制系統
- 支援大型專案協作 (Linux Kernel, GitHub)

## 哲學

- **開放原始碼**: 程式碼應該自由分享
- **實用主義**: 解決問題優先於理論
- **社群協作**: 透過網路協作開發
