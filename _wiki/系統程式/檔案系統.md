# 檔案系統

檔案系統是作業系統管理儲存資料的子系統。

## 檔案系統結構

```
┌────────────┐
│   超級區塊  │  檔案系統資訊
├────────────┤
│   i-node   │  檔案元資料
├────────────┤
│   資料區   │  實際檔案內容
└────────────┘
```

### i-node

i-node 儲存檔案資訊：
- inode 編號
- 檔案類型與權限
- 檔案大小
- 區塊指標
- 時間戳記

## Linux 檔案類型

| 類型 | 說明 |
|------|------|
| 普通檔案 | 一般資料檔案 |
| 目錄 | 資料夾 |
| 裝置檔案 | /dev/tty |
| 管道 | 程序間通訊 |
| 連結 | 捷徑 |

## 一切皆檔案

Linux 的核心哲學：所有 I/O 都以檔案處理。
