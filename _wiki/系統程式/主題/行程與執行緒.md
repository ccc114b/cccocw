# 行程與執行緒

行程是正在執行的程式，執行緒是行程內的執行單位。

## 行程 (Process)

### 行程結構

```
┌─────────────────────┐
│   行程控制區塊 (PCB) │
├─────────────────────┤
│   虛擬位址空間       │
└─────────────────────┘
```

### 行程建立

```c
pid_t pid = fork();
if (pid == 0) {
    // 子行程
    execl("/bin/ls", "ls", "-l", NULL);
}
```

## 執行緒 (Thread)

### 執行緒 vs 行程

| 特性 | 行程 | 執行緒 |
|------|------|--------|
| 資源 | 獨立 | 共享 |
| 建立成本 | 高 | 低 |
| 通訊 | 需要 IPC | 直接共享 |

### pthread 使用

```c
pthread_t tid;
pthread_create(&tid, NULL, func, NULL);
pthread_join(tid, NULL);
```

## 同步機制

### 互斥鎖 (Mutex)

```c
pthread_mutex_lock(&lock);
// 臨界區
pthread_mutex_unlock(&lock);
```

### 訊號量 (Semaphore)

```c
sem_wait(&sem);  // P 操作
// 臨界區
sem_post(&sem);  // V 操作
```
