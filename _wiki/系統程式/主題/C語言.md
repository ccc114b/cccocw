# C語言

C 語言是一種高效、低階的程式語言，廣泛用於系統程式設計。

## 特性

- **低階存取**: 直接記憶體操作指標
- **高效**: 接近硬體的執行效率
- **可移植**: 標準庫確保可移植性
- **簡潔**: 語法簡單，關鍵字少

## 基本語法

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

## 指標

指標是 C 語言的核心：

```c
int x = 10;
int *p = &x;   // p 指向 x
*p = 20;      // 修改 x 的值
```

## 記憶體配置

```c
// 靜態配置
int arr[100];

// 動態配置
int *p = malloc(100 * sizeof(int));
free(p);
```
