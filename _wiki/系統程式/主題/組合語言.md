# 組合語言

組合語言是低階的機器碼表示，使用助記符號代表指令。

## 組合語言 vs 機器碼

| 層次 | 說明 | 範例 |
|------|------|------|
| 高階語言 | 人類可讀 | `a = b + c` |
| 組合語言 | 助記符號 | `add a, b, c` |
| 機器碼 | 二進位 | `0000 1001 ...` |

## RISC-V 組合語言

### 基本指令

```assembly
# 算術
add a0, a1, a2      # a0 = a1 + a2
sub a0, a1, a2      # a0 = a1 - a2
mul a0, a1, a2      # a0 = a1 * a2

# 載入/儲存
lw a0, 0(sp)        # a0 = mem[sp+0]
sw a0, 0(sp)        # mem[sp+0] = a0

# 分支
beq a0, a1, label   # if (a0 == a1) goto label
bne a0, a1, label   # if (a0 != a1) goto label
```

## 函式呼叫

```assembly
# 呼叫 foo(1, 2)
# a0 = 1, a1 = 2
call foo
# 返回值在 a0

foo:
    addi sp, sp, -16    # 分配堆疊空間
    sd ra, 8(sp)        # 保存返回位址
    # ... 函式本體 ...
    ld ra, 8(sp)        # 恢復返回位址
    addi sp, sp, 16     # 釋放堆疊空間
    ret                 # 返回
```
