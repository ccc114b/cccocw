# 同步與通訊

行程間的同步與通訊是作業系統的重要議題。

## 臨界區

多個行程同時存取共享資源的區域。

```c
// 競爭條件
counter++;  // 讀-修改-寫不是原子操作
```

## 同步機制

| 機制 | 說明 |
|------|------|
| 互斥鎖 | 保護臨界區 |
| 訊號量 | 計數器，控制資源數量 |
| 條件變數 | 等待特定條件 |
| 屏障 | 等待所有執行緒到齊 |

## 程序間通訊 (IPC)

| 方法 | 說明 |
|------|------|
| 管道 | 單向資料流 |
| 訊息佇列 | 訊息傳遞 |
| 共享記憶體 | 共用資料區域 |
| Socket | 網路通訊 |

## 死結

四個必要條件：
1. 互斥：資源只能一個使用
2. 占有並等待：已取得資源並等待其他資源
3. 不可搶奪：資源不能強制搶奪
4. 循環等待：形成等待環
