# RISC-V

RISC-V 是一個開放的、精簡的指令集架構 (ISA)。

## 特點

- **開放**: 免費使用，無授權費用
- **模組化**: 基本指令 + 擴展 (I, M, F, D, C)
- **RISC**: 精簡指令集，指令數量少
- **64-bit**: RV64GC 是常見的 64-bit 配置

## 暫存器 (RV64GC)

| 暫存器 | 別名 | 用途 |
|--------|------|------|
| x0 | zero | 永遠為 0 |
| x1 | ra | 返回位址 |
| x2 | sp | 堆疊指標 |
| x8 | s0/fp | 框架指標 |
| x10-x17 | a0-a7 | 函式參數/返回值 |

## 指令類別

- **運算**: add, sub, mul, div, and, or, xor
- **載入/儲存**: lw, ld, sw, sd
- **分支**: beq, bne, blt, bge, j, jal, jalr
