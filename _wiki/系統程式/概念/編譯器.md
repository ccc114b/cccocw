# 編譯器

編譯器是將高階語言翻譯為機器碼的系統軟體。

## 核心功能

1. **詞法分析 (Lexer)**: 將原始碼轉換為 Token 序列
2. **語法分析 (Parser)**: 將 Token 序列轉換為抽象語法樹 (AST)
3. **語意分析 (Semantic)**: 進行型態檢查、作用域檢查
4. **中間表示式 (IR)**: 生成平台無關的中間碼
5. **最佳化 (Optimizer)**: 進行各種最佳化轉換
6. **目的碼生成 (CodeGen)**: 生成目標機器碼

## 編譯器架構

```
原始碼
    │
    ├─> 詞法分析 → Token
    ├─> 語法分析 → AST
    ├─> 語意分析 → 語意檢查
    ├─> IR 生成 → 中間碼
    ├─> 最佳化 → 最佳化 IR
    └─> 目的碼 → 機器碼
```
