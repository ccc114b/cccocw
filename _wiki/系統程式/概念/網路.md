# 網路

網路程式設計涉及 TCP/IP 協定與 Socket 程式設計。

## TCP/IP 模型

```
應用層     │ HTTP, DNS, SSH
傳輸層     │ TCP, UDP
網路層     │ IP, Router
連結層    │ Ethernet, WiFi
```

## TCP vs UDP

| 特性 | TCP | UDP |
|------|-----|-----|
| 連線 | 需要 | 無 |
| 可靠性 | 可靠 | 不可靠 |
| 順序 | 保證 | 不保證 |
| 速度 | 較慢 | 較快 |

## Socket 程式設計

### TCP 伺服器

```c
int server = socket(AF_INET, SOCK_STREAM, 0);
bind(server, (struct sockaddr*)&addr, sizeof(addr));
listen(server, 10);
int client = accept(server, NULL, NULL);
read(client, buf, sizeof(buf));
write(client, "OK", 2);
close(client);
```
