# 強化學習

強化學習是讓智慧體透過與環境互動學習最佳策略。

## 核心概念

- **智慧體 (Agent)**: 學習者
- **環境 (Environment)**: 互動對象
- **狀態 (State)**: 環境的描述
- **動作 (Action)**: 智慧體的選擇
- **獎賞 (Reward)**: 動作的回饋
- **策略 (Policy)**: 狀態到動作的映射

## 學習流程

```
狀態 → 動作 → 環境 → 新狀態 + 獎賞 → 更新策略
```

## 常見演算法

### 基於價值
- Q-Learning
- Deep Q-Network (DQN)

### 基於策略
- Policy Gradient
- REINFORCE
- PPO (Proximal Policy Optimization)

### 基於模型
- Monte Carlo Tree Search
