# 管線處理

管線處理是一種提高 CPU 效率的技術，將指令執行分為多個階段。

## 基本概念

將指令執行分為多個階段：
1. IF (擷取)
2. ID (解碼)
3. EX (執行)
4. MEM (記憶體)
5. WB (寫回)

## 管線示意

```
時鐘:  T1   T2   T3   T4   T5   T6   T7
I1:   IF   ID   EX   MEM  WB
I2:       IF   ID   EX   MEM  WB
I3:           IF   ID   EX   MEM  WB
I4:               IF   ID   EX   MEM  WB
```

## 管線問題

### 結構危險 (Structural Hazard)
- 硬體資源不足

### 資料危險 (Data Hazard)
- RAW (讀後寫)
- WAR (寫後讀)
- WAW (寫後寫)

### 控制危險 (Control Hazard)
- 分支預測失敗
