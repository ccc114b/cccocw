# 暫存器

暫存器是 CPU 內部的高速儲存單元，用於暫時儲存資料與指令。

## 基本暫存器

### 通用暫存器
- 用於暫時存放資料
- RISC-V: x0-x31 (a0-a7, t0-t6, s0-s11)
- x86: eax, ebx, ecx, edx, ...

### 特殊暫存器
- **PC (Program Counter)**: 下一條指令位址
- **IR (Instruction Register)**: 當前執行指令
- **MAR (Memory Address Register)**: 記憶體位址
- **MDR (Memory Data Register)**: 記憶體資料

## 暫存器檔案

CPU 內部的暫存器集合：

```
┌─────────────────────────┐
│      通用暫存器         │
│  x0  x1  x2  ...  x31  │
├─────────────────────────┤
│      特殊暫存器         │
│   PC   IR   MAR   MDR   │
└─────────────────────────┘
```
