# CPU架構

CPU (中央處理器) 是計算機的大腦，負責執行指令。

## CPU 基本結構

```
┌─────────────────────────────────────────┐
│              控制單元                    │
│  - 指令解碼                              │
│  - 控制訊號產生                          │
├─────────────────────────────────────────┤
│              運算單元 (ALU)              │
│  - 算術運算                              │
│  - 邏輯運算                              │
├─────────────────────────────────────────┤
│              暫存器檔案                  │
│  - 通用暫存器                            │
│  - 特殊暫存器                            │
├─────────────────────────────────────────┤
│              快取                        │
│  - L1, L2, L3                           │
└─────────────────────────────────────────┘
```

## 指令週期

1. **擷取 (Fetch)**: 從記憶體取得指令
2. **解碼 (Decode)**: 解析指令意義
3. **執行 (Execute)**: 執行運算
4. **存取記憶體 (Memory)**: 讀寫資料
5. **寫回 (Write Back)**: 寫回結果

## 指令集架構

- **CISC**: 複雜指令集 (x86)
- **RISC**: 精簡指令集 (RISC-V, ARM)
