# 指令集架構

指令集架構 (ISA) 定義了 CPU 支援的指令集合。

## 常見 ISA

| ISA | 開發者 | 特性 |
|-----|--------|------|
| x86 | Intel/AMD | CISC，桌上型/筆電 |
| ARM | ARM | RISC，行動裝置 |
| RISC-V | 開放 | RISC，學術/嵌入式 |
| MIPS | MIPS | RISC，網路設備 |

## 指令格式

### R-type (Register)
```
[opcode][rs][rt][rd][shamt][funct]
```

### I-type (Immediate)
```
[opcode][rs][rt][immediate]
```

### J-type (Jump)
```
[opcode][address]
```

## 指令類別

- **資料傳輸**: move, load, store
- **算術邏輯**: add, sub, and, or, xor
- **控制轉移**: branch, jump, call
