# 組合語言 (Assembly Language)

## 概述

組合語言是介於高階語言和機器碼之間的低階語言，每個指令對應一個機器指令，提供對硬體的直接控制能力。

## RISC-V 組合語言範例

```assembly
# 資料段
    .data
msg:    .asciz "Hello, World!\n"

# 程式碼段
    .text
    .globl _start

_start:
    # 寫入標準輸出
    li a7, 64          # syscall: write
    li a0, 1           # file descriptor: stdout
    la a1, msg         # 訊息位址
    li a2, 14          # 訊息長度
    ecall

    # 結束程式
    li a7, 93          # syscall: exit
    li a0, 0           # exit code
    ecall
```

## 常用指令分類

### 算術運算
| 指令 | 說明 | 範例 |
|------|------|------|
| add | 加法 | `add rd, rs1, rs2` |
| sub | 減法 | `sub rd, rs1, rs2` |
| mul | 乘法 | `mul rd, rs1, rs2` |
| div | 除法 | `div rd, rs1, rs2` |
| rem | 取餘 | `rem rd, rs1, rs2` |

### 邏輯運算
| 指令 | 說明 | 範例 |
|------|------|------|
| and | 位元 AND | `and rd, rs1, rs2` |
| or | 位元 OR | `or rd, rs1, rs2` |
| xor | 位元 XOR | `xor rd, rs1, rs2` |
| sll | 左移 | `sll rd, rs1, rs2` |
| srl | 右移 | `srl rd, rs1, rs2` |

### 比較跳轉
```assembly
    # 比較 if (x > y)
    bgt t0, t1, label   # if t0 > t1, jump to label
    
    # 條件執行
    beq a0, zero, else   # if a0 == 0, goto else
    bnez a1, then        # if a0 != 0, goto then
```

### 記憶體存取
```assembly
    # 載入 (Load)
    lw t0, 0(sp)         # 載入 word
    lb t1, 8(a0)         # 載入 byte (signed)
    lbu t2, 8(a0)        # 載入 byte (unsigned)

    # 儲存 (Store)
    sw t0, 0(sp)         # 儲存 word
    sb t1, 8(a0)         # 儲存 byte
```

## 函式呼叫約定 (Calling Convention)

### RISC-V 暫存器用途
| 暫存器 | 用途 | 是否保存者 |
|--------|------|-----------|
| a0-a7 | 函式引數/返回值 | caller |
| t0-t6 | 暫時變數 | caller |
| s0-s11 | 保存的變數 | callee |
| ra | 返回位址 | caller |
| sp | 堆疊指標 | callee |
| gp | 全域指標 | callee |
| tp | 執行緒指標 | callee |

### 函式範例
```assembly
    .text
    .globl sum_array

# int sum_array(int* arr, int n)
# a0 = arr, a1 = n
# 返回值在 a0
sum_array:
    add t0, zero, zero   # i = 0
    add t1, zero, zero   # sum = 0
    
loop:
    bge t0, a1, end     # if i >= n, goto end
    
    # sum += arr[i]
    slli t2, t0, 2      # offset = i * 4
    add t2, a0, t2      # &arr[i]
    lw t3, 0(t2)        # arr[i]
    add t1, t1, t3       # sum += arr[i]
    
    addi t0, t0, 1      # i++
    jal zero, loop      # goto loop
    
end:
    add a0, t1, zero    # return sum
    jalr zero, ra, 0   # return
```

### 堆疊框架
```assembly
# 建立堆疊框架
addi sp, sp, -16     # 分配 16 bytes
sw ra, 0(sp)         # 保存 ra
sw s0, 8(sp)         # 保存 s0

# ... 函式主體 ...

# 釋放堆疊框架
lw ra, 0(sp)         # 恢復 ra
lw s0, 8(sp)         # 恢復 s0
addi sp, sp, 16      # 釋放空間
jalr zero, ra, 0     # 返回
```

## 組合器偽指令

| 偽指令 | 說明 | 展開 |
|--------|------|------|
| li | 載入立即數 | lui + addi |
| la | 載入位址 | auipc + addi |
| mv | 複製暫存器 | addi rd, rs, 0 |
| neg | 取負 | sub rd, zero, rs |
| not | 位元反轉 | xori rd, rs, -1 |

## 效能考量

### 避免載入使用延遲
```assembly
# 不好：依賴鏈
lw t0, 0(a0)
add t1, t0, t2
lw t3, 4(a0)     # 可以與上面的 add 平行執行

# 好：交錯指令
lw t0, 0(a0)
lw t3, 4(a0)
add t1, t0, t2
add t4, t3, t2
```

### 迴圈展開
```assembly
# 展開 4 次以減少分支開銷
loop:
    lw t0, 0(a1)
    lw t1, 4(a1)
    lw t2, 8(a1)
    lw t3, 12(a1)
    add t4, t0, t1
    add t5, t2, t3
    addi a1, a1, 16
    addi a2, a2, -4
    bnez a2, loop
```
