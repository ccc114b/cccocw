# 測試

測試確保軟體品質，分為多個層次與類型。

## 測試金字塔

```
       /\
      /  \      E2E (少)
     /----\     整合 (中)
    /      \    單元 (多)
   /--------\
```

## 測試類型

### 單元測試
測試最小單位（函數、類別）

```javascript
// Jest
test('adds 2 + 2 to equal 4', () => {
    expect(2 + 2).toBe(4);
});
```

### 整合測試
測試多個單元如何協作

### E2E 測試
測試完整使用者流程

```javascript
// Cypress
cy.visit('/')
cy.get('input').type('hello')
cy.contains('Submit').click()
cy.url().should('include', '/result')
```

## TDD 與 BDD

- **TDD**: 測試驅動開發
  紅 → 綠 → 重構

- **BDD**: 行為驅動開發
  使用自然語言描述行為
