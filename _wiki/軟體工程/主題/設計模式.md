# 設計模式

設計模式是軟體設計中常見問題的標準解決方案。

## 創建型模式

### Singleton 單例
```python
class Singleton:
    _instance = None
    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance
```

### Factory 工廠
```python
class DogFactory:
    def create(self): return Dog()
class CatFactory:
    def create(self): return Cat()
```

## 行為型模式

### Observer 觀察者
```python
class Subject:
    def __init__(self):
        self._observers = []
    def attach(self, observer):
        self._observers.append(observer)
    def notify(self):
        for obs in self._observers:
            obs.update()
```

### Strategy 策略
```python
class Strategy:
    def execute(self, data): pass

class QuickSort(Strategy):
    def execute(self, data): # 快速排序
```
