# 版本控制

版本控制系統追蹤檔案變化歷史，支援多人協作。

## Git 基本指令

```bash
# 初始化
git init
git clone url

# 基礎
git add .
git commit -m "message"
git push origin main
git pull origin main

# 分支
git branch feature
git checkout feature
git checkout -b new-branch
git merge feature

# 查看
git status
git log --oneline
git diff
```

## 分支策略

- **Git Flow**: develop, feature, release, hotfix, main
- **GitHub Flow**: feature 分支 + PR
- **Trunk-Based**: 直接 push 到 main
