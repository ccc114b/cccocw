# 卷積神經網路

卷積神經網路 (CNN) 專門處理網格結構資料，如影像。

## 架構

```
輸入 → 卷積層 → 池化層 → 全連接層 → 輸出
```

## 卷積層

```python
import torch.nn as nn

conv = nn.Conv2d(
    in_channels=3,    # RGB
    out_channels=64, # 輸出通道數
    kernel_size=3,   # 3x3 卷積核
    stride=1,
    padding=1
)
```

## 池化層

- **最大池化**: 取區域最大值
- **平均池化**: 取區域平均值

##經典架構

- LeNet (1998)
- AlexNet (2012)
- VGG (2014)
- ResNet (2015)
- EfficientNet (2019)
