# 循環神經網路

循環神經網路 (RNN) 處理序列資料，適合時間序列、自然語言處理。

## 基本結構

```
時間步: t-1    t    t+1
        ──→  ──→  ──→
        ↑    ↑    ↑
        └────┴────┘
        hidden state
```

## 梯度問題

- **梯度消失**: 長期依賴難以學習
- **梯度爆炸**: 訓練不穩定

## 解決方案

### LSTM (Long Short-Term Memory)

```python
import torch.nn as nn

lstm = nn.LSTM(
    input_size=256,
    hidden_size=512,
    num_layers=2,
    batch_first=True
)
```

門控機制：
- 遺忘門: 決定丟棄什麼
- 輸入門: 決定加入什麼
- 輸出門: 決定輸出什麼

### GRU (Gated Recurrent Unit)

- 更新門與重置門
- 比 LSTM 更簡單
