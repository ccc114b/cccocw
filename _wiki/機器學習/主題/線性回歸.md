# 線性回歸

線性回歸是監督式學習的基本方法，建立輸入與連續輸出間的線性關係。

## 模型

```
y = wx + b
```

- w: 權重（斜率）
- b: 偏差（截距）

## 損失函數

均方誤差 (MSE):

```
L = Σ(yᵢ - ŷᵢ)² / n
```

## 正規方程式

```python
# 封閉解
w = (XᵀX)⁻¹Xᵀy
```

## 梯度下降

```python
# 迭代優化
w = w - α * ∂L/∂w
b = b - α * ∂L/∂b
```
