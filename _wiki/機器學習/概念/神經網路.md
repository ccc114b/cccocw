# 神經網路

神經網路是受生物神經元啟發的計算模型。

## 基本結構

```
輸入層 → 隱藏層 → 輸出層
  │         │
  └─────────┘
    反向傳播
```

## 神經元

```
輸入 → 權重 → 總和 → 啟動函數 → 輸出
 x₁ → w₁ ╲
 x₂ → w₂ ─→ Σ → f(Σ) → y
 x₃ → w₃ ╱
```

## 常見啟動函數

| 函數 | 公式 | 特性 |
|------|------|------|
| Sigmoid | σ(x) = 1/(1+e⁻ˣ) | 0-1輸出 |
| ReLU | max(0, x) | 簡單有效 |
| Tanh | tanh(x) | -1到1輸出 |
| Softmax | eᵢ/Σeʲ | 多分類 |

## 訓練

### 前向傳播
輸入 → 隱藏層 → 輸出 → 計算損失

### 反向傳播
損失 → 梯度 → 更新權重
