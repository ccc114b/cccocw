# 深度學習

深度學習使用多層神經網路學習資料的複雜表示。

## 基礎概念

- **神經元**: 加權輸入 + 啟動函數
- **層**: 多個神經元組成
- **網路**: 多層層組成

## 網路類型

| 類型 | 應用 |
|------|------|
| 全連接層 | 分類、回归 |
| 卷積層 | 影像處理 |
| 循環層 | 序列處理 |
| Transformer | NLP、Vision |

## 訓練過程

```python
# PyTorch 範例
import torch.nn as nn

model = nn.Sequential(
    nn.Linear(784, 256),
    nn.ReLU(),
    nn.Linear(256, 10)
)

criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters())

for epoch in range(10):
    for images, labels in train_loader:
        outputs = model(images)
        loss = criterion(outputs, labels)
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
```
