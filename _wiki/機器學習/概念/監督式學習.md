# 監督式學習

監督式學習使用標記資料進行訓練，學習輸入到輸出的映射。

## 類型

### 回歸 (Regression)
- 輸出為連續值
- 範例: 房價預測、溫度預測

### 分類 (Classification)
- 輸出為類別
- 二分類: 是/否
- 多分類: 類別 A、B、C

## 訓練流程

```
訓練資料 → 模型 → 預測 → 損失函數 → 最佳化 → 更新參數
                  ↑                    ↓
                  └────── 迭代 ────────┘
```

## 常見演算法

- 線性回歸
- 邏輯回歸
- 支持向量機 (SVM)
- 決策樹
- 隨機森林
- 神經網路
