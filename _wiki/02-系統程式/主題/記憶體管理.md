# 記憶體管理

記憶體管理是作業系統的核心功能之一。

## 基本概念

### 連續配置

- **首次適應**: 配置第一個足夠空間
- **最佳適應**: 配置最小足夠空間
- **最差適應**: 配置最大空間

問題：外部碎片

### 分頁 (Paging)

將記憶體分為固定大小的頁面：

```
虛擬位址: | 頁號 | 頁偏移 |
          ─────── ───────
          16 bits  12 bits (4KB 頁面)
```

### 頁表

頁表映射虛擬頁到實體頁框：
```
虛擬頁 0 → 實體頁框 3
虛擬頁 1 → 實體頁框 7
```

## 虛擬記憶體

允許程式使用比實際記憶體更大的位址空間。

### 置換演算法

| 演算法 | 說明 |
|--------|------|
| FIFO | 最早進入的先置換 |
| LRU | 最近最少使用 |
| OPT | 未來最久不使用 |
