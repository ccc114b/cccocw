# 雲端技術

雲端運算與容器化技術是現代部署的基礎。

## 服務模型

| 模型 | 說明 | 範例 |
|------|------|------|
| IaaS | 基礎設施 | EC2, GCP VM |
| PaaS | 平台 | Heroku |
| SaaS | 軟體 | Gmail |

## Docker

容器化平台:

```bash
# 建構映像檔
docker build -t myapp .

# 執行容器
docker run -p 8080:80 myapp
```

## Kubernetes

容器編排系統:

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: myapp
spec:
  replicas: 3
```
