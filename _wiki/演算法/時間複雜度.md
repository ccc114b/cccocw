# 時間複雜度

時間複雜度描述演算法執行時間隨輸入規模增長的趋势。

## 大O標記法

常見複雜度排序：

```
O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2ⁿ)
```

## 常見複雜度

| 複雜度 | 名稱 | 範例 |
|--------|------|------|
| O(1) | 常數 | 陣列存取 |
| O(log n) | 對數 | 二分搜尋 |
| O(n) | 線性 | 線性搜尋 |
| O(n log n) | 線性對數 | 合併排序 |
| O(n²) | 平方 | 氣泡排序 |
| O(2ⁿ) | 指數 | 暴力搜尋 |

## 分析方法

1. 忽略常數係數
2. 忽略低階項
3. 關注最壞情況
