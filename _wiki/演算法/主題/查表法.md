# 查表法 (Lookup Table)

## 概述

查表法是將預先計算好的結果存入表格，运行时直接查表取得答案，避免重複計算。

## 原理

```
輸入 → 查表 → 輸出
```

預先計算所有可能的輸入對應輸出，空間換取時間。

## 常見應用

### 費波那契數列

```python
# 預先計算表
FIB_TABLE = {
    0: 0, 1: 1, 2: 1, 3: 2, 4: 3, 5: 5,
    6: 8, 7: 13, 8: 21, 9: 34, 10: 55
}

def fib_lookup(n):
    if n in FIB_TABLE:
        return FIB_TABLE[n]
    return None  # 超出範圍
```

### 三角函數表

```python
import math

SIN_TABLE = {i: math.sin(math.radians(i)) for i in range(0, 361)}

def sin_lookup(degrees):
    return SIN_TABLE[degrees % 360]
```

### 位運算快速版

```python
POW2 = [1 << i for i in range(32)]

def is_power_of_two(n):
    return n in POW2

def nearest_power_of_two(n):
    for p in POW2:
        if p >= n:
            return p
    return POW2[-1]
```

## 查表法 vs 計算

| 特性 | 查表法 | 線上計算 |
|------|--------|----------|
| 時間複雜度 | O(1) | O(n) 或更高 |
| 空間複雜度 | O(k) | O(1) |
| 適用場景 | 離散、小範圍 | 連續、大範圍 |

## 應用場景

1. **硬體加速**：FPGA/ASIC 中的常數表
2. **密碼學**：S-Box（如 AES 的替換盒）
3. **遊戲開發**：預渲染光影表、動畫幀
4. **科學計算**：特殊函數表
