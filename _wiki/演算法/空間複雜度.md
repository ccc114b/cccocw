# 空間複雜度

空間複雜度描述演算法使用記憶體空間隨輸入規模增長的趋势。

## 分析方法

```python
# O(n) 空間
def create_array(n):
    return [0] * n

# O(1) 空間
def sum_array(arr):
    total = 0  # 只需要一個變數
    for x in arr:
        total += x
    return total
```

## 遞迴空間

```python
# O(n) 遞迴深度
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)
```
